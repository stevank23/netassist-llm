# NetAssist-LM

NetAssist-LM is a networking-focused large language model portfolio project.

The goal of this project is to:
- Fine-tune an open-source LLM on networking Q&A
- Ground answers using Retrieval-Augmented Generation (RAG)
- Deploy an interactive troubleshooting assistant for network engineers

This repository contains:
- Data preparation scripts
- QLoRA fine-tuning pipeline
- Vector database (RAG) construction
- Gradio-based demo application

> This project is built entirely using cloud-based tools (GitHub, Google Colab, Hugging Face).
